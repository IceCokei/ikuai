## 功能

## 依赖
🐉 - 依赖管理 - Python3 - 添加依赖 - python-telegram-bot

## 使用

1. 将压缩包文件下载到本地解压
2. 将四个程序拖入🐉scripts目录
3. 🐉添加任务main.py
4. 更改ikuai.ini 文件
  
- [host ] ikuai地址
- [username ] ikuai账号
- [password ] ikuai密码

5.运行文件等待重播

- 等待重播 详情看日志 默认30s等待时间
- 目前仅适配ADSL/PPPoE拨号模式
